# README

This application is an analogue of the system of questions/answers StackOverflow.

You can see this app there: http://165.22.70.198/

Used technologies:
* Ruby version 2.5.3
* Rails 5.2.3
* PostgreSQL
* Devise
* OAuth - GitHub
* ActiveStorage with AWS S3
* CanCanCan
* SideKiq
* Redis
* SphinxSearch
* Capistrano
* Rest API
* Ajax with ujs and jQuery
* ActionCable
* RSpec & Capybara
